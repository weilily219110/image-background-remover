# 🎯 Image Background Remover — MVP 需求文档

> **版本**：v1.0  
> **日期**：2026-03-23  
> **状态**：初稿

---

## 一、项目概述

| 项目 | 内容 |
|------|------|
| **产品名称** | BG Remover（暂定） |
| **核心功能** | 一键去除图片背景，支持 PNG 透明输出 |
| **目标用户** | 电商卖家、设计师、摄影师、社交媒体运营 |
| **技术方案** | Cloudflare Worker + Remove.bg API |
| **存储需求** | 无（纯内存处理，流式返回） |
| **上线目标** | 2 周内上线 MVP，验证市场需求 |

---

## 二、用户故事 & 功能需求

### 2.1 核心用户故事

```
作为用户，
我上传一张图片（最大 10MB，支持 JPG/PNG/WebP），
系统自动去除背景并返回透明 PNG，
我可以下载结果。
```

### 2.2 功能清单

#### 🔴 P0 — 必须有（MVP）

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 单图上传 | 用户拖拽或选择本地图片 | 支持 JPG/PNG/WebP ≤ 10MB |
| 背景移除 | 调用 Remove.bg API 去除背景 | 返回透明 PNG |
| 图片下载 | 用户下载处理结果 | 点击按钮即可下载 `result.png` |
| 错误提示 | 上传失败 / API 超时时友好提示 | 显示具体错误文案 |

#### 🟡 P1 — 第一版迭代

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 预览对比 | 原始图 vs 处理后图同屏对比 | 左右或上下分屏展示 |
| 历史记录（前端） | 本地 localStorage 缓存最近 5 张 | 刷新页面可看 |
| 多图处理 | 支持批量上传最多 5 张 | 串行处理，显示进度 |

#### 🟢 P2 — 差异化探索

| 功能 | 描述 | 备注 |
|------|------|------|
| AI 生成背景 | 去除背景后，AI 生成新背景 | 可选配色 / 场景 |
| 移动端适配 | 独立移动端页面 | 支持手机使用 |
| Chrome 插件 | 右键一键去背景 | 扩大使用场景 |

## 三、非功能性需求

### 3.1 性能

| 指标 | 目标 |
|------|------|
| 页面加载（LCP） | < 2s |
| API 响应时间（P95） | < 8s（取决于 Remove.bg） |
| 并发支持 | Worker 1000 req/s 不崩溃 |

### 3.2 安全

- API Key 存储于 Cloudflare Secrets
- Worker 仅允许指定域名 CORS
- 上传文件校验类型（MIME + 扩展名双重校验）
- 单文件大小限制 10MB

### 3.3 成本估算（首年）

| 项目 | 单价 | 用量假设 | 月成本 |
|------|------|---------|-------|
| Cloudflare Worker | 免费 100k req/天 | 500 req/天 | $0 |
| Remove.bg API | $0.05/张 | 500 张/月 | $25 |
| 域名 + SSL | ~$10/年 | — | ~$1 |
| **合计** | | | **~$26/月** |

> ⚠️ Remove.bg 免费额度：500 张/月（需要信用卡）
